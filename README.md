# 🍃 Forest Finance (森林風記帳 App)

具備極簡美學與「輕遊戲化」體驗的純前端網頁記帳 App。支援流暢的 SPA 體驗、圖表分析與多國語系，免後端伺服器即可直接部署於 GitHub Pages 免費託管。

## ✨ 特色亮點

- **純前端 架構**：無伺服器與資料庫，透過 `localStorage` 搭配各別 User ID 安全地在一台設備上隔離儲存帳務資料。
- **無縫導覽體驗**：流暢的底部導覽列快速切換日曆帳本、資產總覽與設定，體驗宛如原生手機 App。
- **視覺圖表分析**：內建 Chart.js，全自動將消費紀錄繪製成精美的支出圓餅圖。
- **多語系引擎 (i18n)**：內建多國對應，一鍵切換中、英介面；自帶動態各國貨幣符號 (TWD/USD/JPY 等)。
- **微遊戲化設計**：左上角配有「連續記帳火焰徽章」，紀錄當下再附帶療癒的落葉特效與音效回饋，把記帳變成有趣習慣！

## 🛠 技術棧
HTML5、CSS3、Vanilla JS、Chart.js
